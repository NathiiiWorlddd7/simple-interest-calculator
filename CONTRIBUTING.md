# Contributing

Thanks for your interest in contributing! Please follow these guidelines to help us keep the project healthy and welcoming.

## Getting Started
- Ensure you have Java 17 installed.
- Clone the repository and create a new branch for your changes.

## Development
- Keep changes focused and small; open separate pull requests for unrelated changes.
- Write clear commit messages (imperative mood, short summary line).
- Include tests where applicable or provide examples showing behavior.

## Code Style
- Favor clear, readable code and meaningful names.
- Validate inputs and fail fast with helpful messages.

## Running Locally
- Compile: `javac src/SimpleInterestCalculator.java`
- Run: `java -cp src SimpleInterestCalculator 1000 5 2`

## Pull Requests
1. Fork the repo and create your branch from `main`.
2. Ensure your code builds and runs without errors.
3. Update docs if behavior or usage changes.
4. Open a PR with a clear description of what and why.

## Reporting Issues
- Use the issue tracker with a descriptive title.
- Include steps to reproduce, expected vs. actual results, and environment details.

## Code of Conduct
By participating, you agree to abide by the [Code of Conduct](CODE_OF_CONDUCT.md).
